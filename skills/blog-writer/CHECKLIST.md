# 블로그 글 검수 체크리스트

## 사용 방법

글 작성 완료 후, 아래 체크리스트를 하나씩 확인합니다.
모든 항목을 통과해야 발행 가능합니다.

---

## 1. 제목 검수

### 필수 항목

- [ ] 제목이 본문 내용을 정확히 반영하는가?
- [ ] 메인 키워드가 제목에 포함되어 있는가?
- [ ] 제목 길이가 50자 이내인가?
- [ ] 초보자가 봐도 이해할 수 있는 제목인가?

### 품질 항목

- [ ] 클릭하고 싶은 제목인가?
- [ ] 과장이나 낚시가 아닌가?
- [ ] 검색 결과에서 경쟁 글과 차별화되는가?

---

## 2. 구조 검수

### 필수 항목

- [ ] 들어가며(서론)가 있는가?
- [ ] 목차/단계가 명확히 구분되어 있는가?
- [ ] 마무리(결론)가 있는가?
- [ ] 논리적 흐름이 자연스러운가?

### 품질 항목

- [ ] 한 섹션이 너무 길지 않은가? (스크롤 3번 이내)
- [ ] 적절한 소제목으로 나뉘어 있는가?
- [ ] 시각적으로 읽기 편한 구조인가?

---

## 3. 초보자 친화성 검수

### 필수 항목

- [ ] 전문 용어 첫 등장 시 설명이 있는가?
- [ ] 한 단계에 한 가지 행동만 있는가?
- [ ] "왜" 하는지 설명이 있는가?
- [ ] 선수 지식 요구사항이 명시되어 있는가?

### 품질 항목

- [ ] 개발을 모르는 사람도 따라할 수 있는가?
- [ ] 중간에 막히는 부분이 없는가?
- [ ] 친근하고 격려하는 어조인가?

---

## 4. 코드/기술 검수

### 필수 항목

- [ ] 모든 코드가 실제로 동작하는가? (직접 테스트)
- [ ] 코드 블록에 언어가 명시되어 있는가? (```python)
- [ ] 복사해서 바로 실행 가능한가?
- [ ] 실행 결과가 함께 제공되는가?

### 품질 항목

- [ ] 코드에 적절한 주석이 있는가?
- [ ] 불필요한 코드가 없는가?
- [ ] 최신 버전/방법을 사용했는가?

### 오류 처리

- [ ] 자주 발생하는 오류가 안내되어 있는가?
- [ ] 오류 해결 방법이 제공되는가?

---

## 5. 정보 정확성 검수

### 필수 항목

- [ ] 공식 문서와 교차 검증했는가?
- [ ] 버전 정보가 최신인가? (1년 이내)
- [ ] 링크가 모두 유효한가?
- [ ] 잘못된 정보가 없는가?

### 품질 항목

- [ ] 출처가 필요한 정보에 출처가 있는가?
- [ ] 의견과 사실이 구분되어 있는가?

---

## 6. 문법/맞춤법 검수

### 필수 항목

- [ ] 맞춤법 오류가 없는가?
- [ ] 띄어쓰기가 올바른가?
- [ ] 문장이 자연스러운가?

### 품질 항목

- [ ] 일관된 어조를 사용했는가? (~하세요 / ~합니다)
- [ ] 불필요한 반복이 없는가?
- [ ] 문장이 너무 길지 않은가? (40자 이내 권장)

---

## 7. SEO 검수

### 필수 항목

- [ ] 메타 설명(description)이 작성되었는가?
- [ ] 메인 키워드가 본문에 자연스럽게 포함되었는가?
- [ ] 이미지에 alt 텍스트가 있는가?

### 품질 항목

- [ ] 내부 링크가 포함되어 있는가?
- [ ] 외부 권위 있는 링크가 있는가?
- [ ] URL이 키워드를 포함하는가?

---

## 8. 최종 점검

### 발행 전 필수

- [ ] 전체 글을 처음부터 끝까지 읽어보았는가?
- [ ] 모바일에서 읽기 편한가?
- [ ] 이 글로 독자의 목표를 달성할 수 있는가?

---

## 검수 결과 템플릿

검수 완료 후 다음 형식으로 보고:

```markdown
## 검수 결과

**글 제목**: [제목]
**검수 일시**: [날짜]

### 통과 항목
- ✅ 제목 검수 (7/7)
- ✅ 구조 검수 (6/6)
- ...

### 수정 필요 항목
- ⚠️ [항목명]: [구체적인 문제점]
- ⚠️ [항목명]: [구체적인 문제점]

### 권장 개선 사항
- 💡 [개선 제안]

### 최종 판정
- [ ] 발행 가능
- [ ] 수정 후 발행
- [ ] 대폭 수정 필요
```

---

## 빠른 검수 (시간이 없을 때)

최소한 이것만이라도:

1. [ ] 제목이 내용을 반영하는가?
2. [ ] 코드가 동작하는가?
3. [ ] 초보자가 따라할 수 있는가?
4. [ ] 맞춤법 오류가 없는가?
5. [ ] 처음부터 끝까지 읽어보았는가?
