## 초보자 친화 작성 원칙

### 1. 전문 용어 처리

**규칙**: 전문 용어 첫 등장 시 반드시 설명

```markdown
❌ "터미널에서 npm install을 실행하세요."

✅ "터미널(명령어를 입력하는 검은 화면)을 열고,
    npm install(필요한 파일들을 자동으로 다운로드하는 명령어)을
    입력하세요."
```

### 2. 단계별 설명

**규칙**: 한 단계에 한 가지 행동만

```markdown
❌ "Node.js를 설치하고 터미널을 열어 npm init을 실행한 후
    package.json을 수정하세요."

✅ "### Step 1: Node.js 설치
    1. nodejs.org에 접속합니다
    2. LTS 버전을 클릭합니다
    3. 다운로드된 파일을 실행합니다

    ### Step 2: 설치 확인
    ..."
```

### 3. 코드 블록 작성

**규칙**: 복사 가능 + 실행 결과 포함

````markdown
다음 명령어를 입력하세요:

```bash
python --version
```

실행하면 이런 결과가 나옵니다:

```
Python 3.11.5
```

> 버전 숫자가 다를 수 있어요. 3.x로 시작하면 정상입니다!
````

### 4. 시각 자료 활용

**규칙**: "클릭", "선택" 등의 동작에는 스크린샷 또는 설명

```markdown
**VSCode 설치 화면**

1. "Download for Windows" 버튼 클릭
   (화면 중앙의 큰 파란색 버튼입니다)

2. 다운로드가 완료되면 파일 실행
   (보통 "다운로드" 폴더에 저장됩니다)
```

### 5. 예상 결과 미리 보여주기

**규칙**: 독자가 "이게 맞나?" 불안하지 않도록

```markdown
이 단계를 완료하면 다음과 같은 화면이 보입니다:

[예상 결과 설명 또는 스크린샷]

> 만약 다른 화면이 보인다면 [문제 해결 섹션]을 확인하세요.
```

## 코드 예제 작성 규칙

### 완전한 예제 제공

```markdown
❌ 일부만 보여주기:
```python
print(result)
```

✅ 전체 코드 제공:
```python
# 파일명: hello.py
# 첫 번째 Python 프로그램

message = "Hello, World!"
print(message)

# 실행 방법: python hello.py
# 예상 결과: Hello, World!
```
```

### 주석으로 설명

```python
# 1. 변수 만들기 (데이터를 담는 상자)
name = "홍길동"

# 2. 화면에 출력하기
print(name)  # 결과: 홍길동

# 3. 변수 내용 바꾸기
name = "김철수"
print(name)  # 결과: 김철수
```

### 오류 예시 포함

**원인:** requests 라이브러리가 설치되지 않았습니다.

**해결 방법:**
```bash
pip install requests
```

## 문단 작성 규칙

### 길이

- 한 문단: 3-4문장 이하
- 한 문장: 40자 이하 권장

### 어조

- 친근하지만 정확하게
- "~하세요" (명령형) 또는 "~합니다" (설명형) 일관되게 사용
- "!" 는 격려할 때만 사용

```markdown
❌ "이제 개발환경 세팅이 완료되었습니다!!!"
✅ "개발 환경 설정이 완료되었습니다. 수고하셨어요!"
```

### 강조

- **굵게**: 핵심 개념, 중요한 버튼/메뉴명
- `코드`: 명령어, 파일명, 코드
- > 인용: 팁, 주의사항

## 섹션별 작성 가이드

### 들어가며 (서론)

```markdown
## 들어가며

Python을 처음 배우시나요?
이 글에서는 Python 설치부터 첫 코드 실행까지 안내해드립니다.

**이 글에서 배울 것:**
- Python 설치하기
- 첫 번째 코드 작성하기
- 코드 실행하기

**예상 소요 시간:** 약 10분

**필요한 것:** 컴퓨터 (Windows/Mac)
```

### 본문 (단계별)

```markdown
## Step 1: Python 다운로드

### 1-1. 공식 사이트 접속

웹 브라우저를 열고 주소창에 입력하세요:

```
https://www.python.org
```

### 1-2. 다운로드 버튼 클릭

화면 상단의 **Downloads** 메뉴에 마우스를 올리면
자동으로 추천 버전이 표시됩니다.

> 💡 **팁**: "Download Python 3.x.x" 버튼을 클릭하세요.
> 숫자는 버전 번호로, 최신 버전이면 됩니다.
```
