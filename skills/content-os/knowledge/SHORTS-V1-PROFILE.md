# 쇼츠 스크립트 v1 프로필 (지식파일)

**버전**: v1 (18-20초)
**글자수**: 80-120자
**줄 수**: 7-9줄

---

# PART 1: REF (전략)

## 1. Value Proposition

### 타겟
- 연령: 10-30대
- 특성: 짧은 콘텐츠 소비자, 스크롤 빠름
- 니즈: 빠른 정보 + 재미

### 제공 가치
```
"몰랐던 정보 + 반전 + 웃음"을 15-20초에 압축
```

### 차별점
- 정보 전달만 ❌
- 정보 + 감정 동시 자극 ✅

---

## 2. Psychological Flow

### 감정 곡선
```
호기심 → 집중 → 긴장 → 반전/해소
  ↑        ↑       ↑        ↑
 훅      본문   클라이맥스  마무리
```

### 단계별 심리
| 단계 | 시간 | 목표 감정 | 방법 |
|------|------|----------|------|
| 훅 | 0-2초 | 호기심 | "뭐지?" 유발 |
| 본문 | 2-16초 | 집중+긴장 | 정보+서스펜스 |
| 마무리 | 16-20초 | 만족/웃음 | 반전/공감 |

---

## 3. Strategic Intent

### 목표 행동
1. 끝까지 시청 (시청 완료율)
2. 좋아요/저장
3. 댓글 반응
4. 공유

### 핵심 메시지 공식
```
"[대상]이 [상황]이라는데, [반전/결론]"
```

### 성공 기준
- 시청 완료율 > 70%
- 좋아요율 > 5%

---

## 4. 콘텐츠 유형

### 유형 A: 트렌드/이슈
```
공식: "요즘 ~라는데" → 상황 설명 → 반응 소개 → 위트 마무리
심리: 트렌드 뒤처짐 불안 → 정보 획득 → 안심
```

### 유형 B: 반전형
```
공식: "~인 줄 알았는데" → 정보 → "그러나 진짜 반전은?" → 반전 공개
심리: 기대 → 긴장 → 놀람
```

### 유형 C: 리스트형
```
공식: "~한 N가지" → 1번 → 2번 → N번 → 감탄 마무리
심리: 정리된 정보 욕구 충족
```

### 유형 D: 비교형
```
공식: "A vs B" → A 설명 → B 설명 → 결과/반응
심리: 판단 욕구 충족
```

### 유형 E: 국뽕형
```
공식: "세계가 놀란" → 배경 → 한국 기술/성과 → 자부심 마무리
심리: 소속감 + 자부심
```

# PART 2: RFE (실행규칙)

## 1. Structure Rules

### 전체 스펙
| 항목 | 규칙 |
|------|------|
| 길이 | 18-20초 |
| 글자수 | 100-130자 |
| 줄 수 | 12-14줄 |
| 한 줄 | 5-18자 (가변) |

### 섹션 구성
```
┌─────────────────────────────────┐
│ 훅 (1줄)       │ 10%  │ ~12자  │
│ 본문 (10-12줄)  │ 80%  │ ~100자 │
│ 마무리 (1줄)   │ 10%  │ ~12자  │
└─────────────────────────────────┘
```

### 줄바꿈 특징
- 짧은 줄(5자)과 긴 줄(18자) 혼합 OK
- 호흡 단위로 자유롭게 끊기
- 정보 나열 시 항목별로 줄바꿈

### 훅 공식 (첫 줄) - 택 1
```
□ "[명사]가 [동사]한다는"
□ "[대상]이 [상황]이라는"
□ "사람들이 [행동]하는 이유"
□ "[특징]으로 [행동]한 사람"
□ "[반전] + [대상] 근황"
```

### 마무리 공식 (끝 줄) - 택 1
```
□ 질문형: "~인가?", "~인데?", "~하려나?"
□ 감탄형: "~였네", "~이구나", "~맞다"
□ 위트형: "야! ~", "~ㅋㅋ", "~ㄷㄷ"
□ 자조형: "나만 ~인가?ㅠㅠ"
```

---

## 2. Tone & Manner

### 격식
| 항목 | 규칙 |
|------|------|
| 말투 | 반말 100% |
| 느낌 | 친구한테 얘기하듯 |

### 허용/금지
```
✅ 허용
- 짧은 문장 (10-15자)
- 구어체 ("~인데", "~거든", "~라고")
- 줄임말 ("걍", "겜", "꿀잼")

❌ 금지
- 딱딱한 표현 ("~입니다", "~하였습니다")
- 긴 수식어
- 존댓말
```

### 감정 표현
| 감정 | 표현 | 사용 위치 |
|------|------|----------|
| 가벼운 웃음 | ㅋㅋ | 마무리 |
| 놀람 | ㄷㄷ | 반전 후 |
| 안타까움 | ㅠㅠ | 공감 유도 |
| 충격 | ?! | 반전 직후 |

---

## 3. Language Rules

### 필수 패턴 (최소 2개 사용)
```
□ "~라는", "~한다는데", "~라고 함"
□ "그러나", "그런데", "의외로"
□ "결국", "심지어", "특히"
□ "알고 보니", "진짜 반전은"
```

### 금지어
```
❌ "~입니다", "~합니다" (딱딱함)
❌ "매우", "정말로", "굉장히" (과장)
❌ "최고의", "완벽한" (신뢰↓)
❌ 설명 없는 전문용어
```

### 숫자 표현
```
✅ "15조 원", "42살" (한글+숫자)
✅ "백만", "천만" (단위 간결)
❌ "150,000,000,000원" (읽기 어려움)
```

---

## 4. Format Rules

### 줄바꿈 규칙
```
✅ 1-2문장마다 줄바꿈
✅ 호흡 단위로 끊기
✅ 한 줄 = 화면 한 장면
✅ 반전 전 줄바꿈으로 텀
```

### 강조 규칙
```
✅ 핵심 단어 앞에 배치
✅ 반전 전 한 줄 띄우기
❌ 볼드/이탤릭 사용 안 함
```

---

## 5. Filter

### ✅ 통과 조건 (전부 만족)
```
□ 글자수 100-130자
□ 줄 수 12-14줄
□ 첫 줄이 호기심 유발
□ 마지막 줄에 반전/위트
□ 반말 사용
□ 정보 + 감정 둘 다 있음
```

### ❌ 차단 조건 (하나라도 해당 시 수정)
```
□ 첫 줄이 설명으로 시작
□ 문장이 3줄 이상 끊김 없이 이어짐
□ 마무리가 밋밋함
□ 정보만 나열 (감정 없음)
□ 존댓말 사용
□ 130자 초과
```

# PART 3: 예시

## Good Example ✅

```
사람들이 생각하는 카카오톡 대체 어플
뇌절 가득한 업데이트로
카톡에 불만이 쌓이자 
업무용으로 깔끔하다는 네이트온부터 
성능은 괜찮지만 이미지 곱창난 텔레그램 
구시대 유물인 버디버디까지 
지하실에 숨어 있던 어플들이 재조명 받는다는데 
그러나 대부분은 편리기능 다 갖춘 네이버가 
노젓기를 기대하는 중이라 함
네이버야 함해주라ㅋㅋ
```

**분석**:
- 글자수: ~100자 ✅
- 훅: "사람들이 생각하는~" (공식 적용) ✅
- 마무리: 위트형 "ㅋㅋ" ✅
- 톤: 반말, 구어체 ✅
- 필수 패턴: "~라는데", "그러나" ✅

---

## Bad Example ❌

```
카카오톡의 최근 업데이트에 대해 많은 사용자들이
불만을 표출하고 있습니다. 이에 따라 대체 
메신저 애플리케이션에 대한 관심이 높아지고 
있는 상황입니다. 네이트온, 텔레그램, 버디버디 
등이 거론되고 있으나, 네이버의 라인 메신저가 
가장 유력한 대안으로 주목받고 있습니다.
```

**문제점**:
- 딱딱함: "~입니다", "~하고 있습니다" ❌
- 존댓말 사용 ❌
- 감정 요소 없음 ❌
- 훅 약함: 설명으로 시작 ❌
- 마무리 밋밋함 ❌
