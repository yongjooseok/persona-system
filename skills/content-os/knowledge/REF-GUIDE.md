# REF 추출 가이드 (지식파일)

## REF란?

**Reference Frame** - 전략/로직 레벨
- "왜 이 결과물이 성공했는가?"
- 판단과 사고의 기준 프레임

---

## 추출 순서

> ⚠️ **RFE 먼저 추출 후 REF 작업** (실행규칙 → 전략 순서)

```
샘플 읽기 → 공통점 발견 → 의도 해석 → REF 문서화
```

---

## 1. Value Proposition 추출

### 질문 리스트
```
□ 이 콘텐츠는 누구를 위한 것인가?
□ 무엇을 제공하는가? (정보/재미/감동/공감)
□ 왜 이게 먹히는가?
□ 경쟁 콘텐츠와 뭐가 다른가?
```

### 추출 방법
1. 샘플들의 공통 타겟 파악
2. 제공하는 가치 유형 분류
3. 차별화 포인트 도출

### 출력 형식
```markdown
## Value Proposition

### 타겟
- 연령: [범위]
- 특성: [특징]
- 니즈: [욕구]

### 제공 가치
- [핵심 가치 한 문장]

### 차별점
- [경쟁 대비 차별화 요소]
```

---

## 2. Psychological Flow 추출

### 질문 리스트
```
□ 시작할 때 어떤 감정을 유발하는가?
□ 중간에 감정이 어떻게 변하는가?
□ 끝날 때 어떤 감정으로 마무리하는가?
□ 왜 끝까지 보게 되는가?
```

### 추출 방법
1. 샘플 각각의 감정 흐름 매핑
2. 공통 패턴 발견
3. 감정 곡선 도식화

### 출력 형식
```markdown
## Psychological Flow

### 감정 곡선
[시작 감정] → [중간 전환] → [최종 감정]

### 단계별 심리
| 단계 | 목표 감정 | 방법 |
|------|----------|------|
| 시작 | [감정] | [방법] |
| 중간 | [감정] | [방법] |
| 끝 | [감정] | [방법] |
```

---

## 3. Strategic Intent 추출

### 질문 리스트
```
□ 이 콘텐츠를 본 후 독자가 뭘 하길 원하는가?
□ 핵심 메시지를 한 문장으로 하면?
□ 성공/실패를 어떻게 측정하는가?
```

### 추출 방법
1. 샘플들의 CTA 분석
2. 공통 메시지 도출
3. 성과 지표 정의

### 출력 형식
```markdown
## Strategic Intent

### 목표 행동
1. [1차 목표]
2. [2차 목표]

### 핵심 메시지
- [한 문장으로]

### 성공 기준
- [측정 가능한 지표]
```

---

## 4. 콘텐츠 유형 분류

### 추출 방법
1. 샘플들을 유사 패턴끼리 그룹핑
2. 각 그룹의 공통 공식 추출
3. 유형별 이름 부여

### 출력 형식
```markdown
## 콘텐츠 유형별 전략

### 유형 A: [이름]
- 공식: [패턴]
- 심리: [흐름]
- 예시: [샘플 번호]
```

---

## 품질 체크

REF 완성 후 확인:
```
□ 샘플 없이 REF만 읽어도 전략이 이해되는가?
□ 새로운 콘텐츠 기획 시 방향을 잡을 수 있는가?
□ "왜?"에 대한 답이 명확한가?
□ 추상적이지 않고 구체적인가?
```

---

## 주의사항

1. **해석 과잉 금지**: 샘플에 있는 것만 추출
2. **구체적으로**: 추상적 표현 피하기
3. **검증 필수**: 추출한 REF로 새 콘텐츠 만들어서 검증
